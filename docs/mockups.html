갤러리 — 2열 + FAB
필터 바 전체를 [필터 ▾] 버튼 하나로 접고 검색만 상시 노출합니다. 하단 탭바는 [홈][갤러리] 둘뿐 — 수업 중에 실제로 오가는 화면은 이 두 개입니다.

목업에 반영된 결정

  • 썸네일 폴백이 기본값 — 회색 빈 박스 대신 주소 해시 기반 그라데이션. OG 자동 추출은 그 위에 얹는 보너스입니다.
  • PIN은 거의 안 보이게 — 같은 브라우저에서는 묻지 않습니다. 게시 직전 확인 한 번으로 오타 잠김을 막습니다.
  • 관리자 모드는 인라인 — 별도 관리자 사이트 없이 같은 화면에 편집 컨트롤이 켜집니다.
  • 공개 동의 체크박스 — 로그인이 없다는 사실을 올리는 순간 한 번은 알려야 합니다.
  • 첫 배포에는 댓글·검색·태그 필터가 없습니다 — 위 목업에는 완성형을 그렸지만, 첫 수업에는 결과물이 0~5개라 필터할 대상이 없습니다. 태그 입력만 첫날부터 켭니다.
